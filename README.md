## Media Assets (Audio & Covers)

This repository does not include full audio tracks or album cover images.

- Audio files (`song/` folder) are excluded due to file size limits and copyright restrictions.
- Cover images (`cover/` folder) are excluded for the same reasons.

To run the project locally:
1. Create a `song/` folder and add your own audio files using the expected filenames.
2. Create a `cover/` folder and add corresponding cover images.
